# owanbe-arena_app
What the product does
Owanbe Arena, is a mobile application for event planning that streamlines the organization process by linking users with vendors, venues, and crucial services for efficient planning. 
Why it matters
Organizing events in Nigeria is frequently a stressful, fragmented experience that heavily depends on personal referrals. Owanbe Arena seeks to solve the problem of chaotic and ineffective event management by providing a centralized platform for planning, booking vendors, and coordinating guests, all customized to fit the unique culture of Nigerian festivities.
Who is it for
- Event planners and individuals organizing personal or corporate events
- Vendors offering catering, decoration, music, photography, and logistics services
- Guests who want to discover and engage with events
Key features
- Vendor marketplace with verified profiles  
- Event budget and checklist manager  
- Direct chat and booking system  
- Ratings and reviews for service providers  
- Smart recommendations based on event type and location
